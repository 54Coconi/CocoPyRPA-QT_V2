欢迎使用CocoPyRPA桌面自动化程序，以下是一些主要功能及说明：

指令功能说明

鼠标控制类指令

指令 描述
鼠标左键点击 模拟鼠标左键点击
鼠标右键点击 模拟鼠标右键点击
鼠标中键点击 模拟鼠标中键点击
鼠标定点移动 鼠标移动至屏幕指定坐标处
鼠标相对移动 鼠标相对当前位置移动指定距离
鼠标定点拖动 鼠标从当前位置拖动到屏幕指定位置
鼠标相对拖动 鼠标相对当前位置拖动指定距离
鼠标竖直滚动 模拟鼠标滚轮竖直滚动
鼠标水平滚动 模拟鼠标滚轮水平滚动

键盘控制类指令

指令 描述
键盘按下 模拟键盘单个按键按下
键盘释放 模拟键盘单个按键释放
键盘敲击 模拟键盘单个按键敲击（按下并释放）
热键组合 模拟键盘多个按键同时按下并释放
键盘输入文本 模拟键盘通过按键输入字符串

图像类指令

指令 描述
图片匹配 截取屏幕全屏并将用户提供的模板图片与之匹配，匹配成功返回模板中心坐标
图片点击 先通过图片匹配找到模板图片的中心位置再将鼠标移动到该位置并点击
文字识别 先截取全屏图像，再识别图像中的文字并返回给定文字匹配结果
文字点击 文字识别成功后点击该文字区域的中心位置

流程控制类指令

指令 描述
等待 等待指定时间
If 判断 通过判断当前"if 判断"指令之前某个已经执行的指令的属性值来决定执行"then"分支还是"else"分支
Loop 循环 将多个指令整体循环执行指定次数

脚本控制类指令

指令 描述
执行 DOS 命令 使用subprocess库执行Dos命令，支持Windows和Unix平台（在Unix平台执行的是shell命令）
执行 Python 脚本 在预设的python环境中执行python代码

子任务类指令

指令 描述
执行子任务 用于加载子任务文件的指令到主任务中

菜单项、工具项说明

操作 快捷键 描述
文件
新建任务目录 Ctrl+Shift+N 在任务列表中新建一个任务目录文件夹
新建任务文件 Ctrl+N 必须在选中了某个任务目录后才能新建一个任务json文件
打开任务 Ctrl+O 打开选择的任务json文件并将任务加载到当前指令编辑区内
保存任务 Ctrl+S 保存当前指令编辑区内的指令到选中的任务json文件中
设置 无 打开软件设置窗口
编辑
撤销 Ctrl+Z 撤销指令编辑区内上一步的修改操作
重做 Ctrl+Y 重做上一步的撤销动作
清空 Ctrl+Delete 清空指令编辑区内的所有指令
视图
属性编辑器 无 控制是否显示属性编辑器停靠窗口
指令操作库 无 控制是否显示指令操作库停靠窗口
运行
运行全部 Shift+F5 从指令编辑器内的第一条指令开始加载指令并从第一条指令开始运行
指定运行 Shift+F6 加载指令编辑器内用户选中的一条指令并运行该指令
从当前运行 Shift+F7 从指令编辑器内的第一条指令开始加载指令并从用户选中的一条指令位置开始往下运行
自动运行 无 打开自动执行管理器界面
工具
截图 Ctrl+Shift+S 打开截图工具用于截取图片，单击可以保存该点坐标
鼠标录制 Ctrl+Shift+M 检测鼠标的基本操作并记录，生成对应指令
键盘录制 Ctrl+Shift+K 检测键盘按键按下和释放事件并记录，生成对应指令
运行Python脚本 无 打开Python代码编辑器，在软件预设的执行环境中执行代码
主题
默认 无 设置软件主题为“默认”主题
深色 无 设置软件主题为“深色“主题
浅色 无 设置软件主题为”浅色“主题
护眼 无 设置软件主题为”护眼“主题
帮助
功能说明 无 打开软件功能说明窗口
关于 无 打开软件关于信息窗口
检查更新 无 检查软件版本更新
